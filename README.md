# ArcPay – Payment + Shopping App on Arc Testnet

Built with official Circle App Kit + Developer Controlled Wallets.

## Setup
1. `npm install`
2. Copy `.env.example` → `.env.local` and fill all keys
3. `npx supabase start` (or link your project)
4. `npm run dev`

Add Arc Testnet to MetaMask:
- RPC: https://rpc.testnet.arc.network
- Chain ID: 5042002
- Symbol: USDC
- Explorer: https://testnet.arcscan.app

Get test USDC: https://faucet.circle.com
